**Project Overview**

This Power BI project analyzes airline customer behavior, booking patterns, loyalty metrics, and satisfaction trends. The dashboard provides insights that help improve customer experience, optimize operations, and support data-driven decision-making.

**Key Features**

* Customer booking and travel trend analysis
* Loyalty tier distribution
* Demographic segmentation
* Customer satisfaction and feedback insights
* Flight frequency and revenue contribution
* Visual KPIs for quick decision-making

**Tools Used**

* Power BI Desktop
* Power Query
* DAX Measures
* Excel / CSV Data Source

**Insights Highlighted**

* Top-performing customer segments
* Behavior differences across age, gender, and loyalty tiers
* Peak travel periods & booking trends
* Satisfaction drivers and complaint trends
